Hexa Path is a simple and fun board-style game developed using Python. This game is a GUI-based project made with Tkinter and PIL where players move their pieces according to dice rolls. The game can be played with friends or against the computer. It is designed to be easy to play and enjoyable for all ages.

## What is Hexa Path?
Hexa Path is a turn-based game where players move their pieces step by step based on dice results. The main goal is to reach the final position before the other players and win the game.

## Hexa Path Project Details
Hexa Path is a GUI-based project built using the tkinter and PIL modules. The game starts with a window where the user can choose the game mode and number of players. After a short delay, the game begins.

The game supports multiple players, and each player has a turn to roll the dice and move their piece accordingly. When a player rolls a 6, they can bring a piece into the game and start moving it on the board. The game continues until one player reaches the final destination and wins.

This project demonstrates how to build interactive games using Python GUI libraries and handle game logic, animations, and user interaction.
